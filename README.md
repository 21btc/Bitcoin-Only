## Build Setup


```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:1111
$ npm run dev

# generate static project (dist folder)
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
